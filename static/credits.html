Many open source libraries, projects, and more make Verdant Shift possible. They are listed here.

Programming Languages Used (for nerds)

NodeJS (JavaScript): Handling server-side computations and requests from clients
HTML: Data sent to client
CSS: HTML page styling
JavaScript: Client side computations, etc.

Server & Backend

Replit & Replit Deployments: Running & deploying server
Express.js (Express): Handling requests sent to server
Replit Object Storage: Backend data storage
Replit Authentication: Login

Parsing & Sending Data

Embedded JavaScript Templates (EJS): Customizing pages sent to client
Cookie Parser: Reading browser cookies from client

Express Middleware

Server-side middleware to parse data, etc. This is part of ExpressJS.
Express.json: Parsing JSON in client requests
Express.static: Serving static files (such as images)
Express 404: Handling requests to pages that don't exist
